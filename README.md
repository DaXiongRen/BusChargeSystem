# BusChargeSystem

## 项目介绍

基于STM32F407+RFID的公交车收费系统

做这个小小的项目过程中参考了很多正点原子的资料，很多东西都是刚开始学习，所以只会实现一些简单的增删改查功能。

此项目对于刚刚学习STM32的还是挺合适的。
